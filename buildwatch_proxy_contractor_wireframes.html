Screen notes — Proxy Audit Trail

Framing
"My visit record" not "Audit trail" — proxy-friendly language, emphasises their protection

BuildWatch — Proxy & Contractor Screens Reference · PRD v1.3 · June 2026
11 screens · 6 proxy (mobile) + 5 contractor (mobile) · Viewport: 390px · Stack: Next.js PWA · Offline queue · PostgreSQL · Prisma · Vercel · DeepSeek AI (owner-side only)
For Antigravity agent use. Proxy screens: start with P2 Home to establish mobile shell. Contractor screens: start with C1 Home. Both share same mobile shell pattern — topbar + content + bottom nav. No sidebar on mobile.
Key rule: AI is invisible to proxy and contractor. GPS is invisible to proxy. Total budget invisible to contractor. These are non-negotiable PRD mandates.
